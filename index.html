Introduction

Balises générales de contenu

Généralement, tout les contenus de type texte sont a mettre dans des balise de type paragraphe. Il n'y a pas que la balise <p> </p>

Il y a deux type de balises qui contiennent du texte, celles avec un "display block" d'autres "inline".

  • Block :
    la balise prend tout l'espace en largeur, et ne peut pas contenir d'autre balise de texte de type block, sauf contre-indication en CSS.
    En revanche, elle peut contenir des balise de texte de type inline
  • Inline :
    ces balises entourent le texte, sans générer de marge, d'espace, sauf contre-indication en CSS.

Balises sur le texte

Il existe des balise inline pour modifier l'apparence du texte.

Par exemple, pour mettre simplement en gras, on utilise la balise <b>, mais si le texte mis en gras a une importance sur la pertinence du contenu niveau SEO, on utilisera la balise <strong>.
Il existe le même comportement pour l'italique : la balise <i> pour simplement l'afficher visuellement, la balise <em>, qui elle ajoute de l'emphase sur le contenu.

Autrefois, pour changer la couleur, la police d'un morceau de texte, on utilisait font, maintenant, on va plutôt utiliser une balise générale inline <span>.
